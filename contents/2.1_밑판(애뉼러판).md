### 시험개소 및 범위
| 구분 | 검사 종류 | 검사량(범위) |
|------|-----------|----------------|
| **애뉼러판<Br>맞대기이음** | **초층 PT →<Br> 용접종료 MT** | **전체 길이** |
|  | ② RT 또는 PAUT + PT/MT | - 끝단 250mm 이내: 전체 중 50% RT 또는 PAUT <br>- 250mm 초과부: PT + MT |
| **밑판<Br>(애뉼러판 없음)** | **PT + MT** | - 옆판 최하단<Br> 중심 기준<br>→ **안쪽 200mm + 바깥쪽 전체 용접부** |
| **밑판 이음부<br>(3매 겹침부등)** | **PT + MT** | **200mm × 3방향** |
| **밑판 용접부<Br>(전 길이의 3%)** | **① MT** 또는 ② 전체 진공시험 | - **전체 길이의 3%<br>- 단, 10m 초과 시 최대 10m**|
| **필렛용접부<br>(밑판 또는 애뉼러판과 옆판 용접부)** | **충수 전, 후 MT** | **전체 길이**|
| **SUMP(물받이)** | **① MT** | **① 맞대기 제외 전체 용접부** |
|  | **② PAUT 또는 RT** | **② 맞대기이음부** |
| **옆판 열영향부** | **MT** | **1단 수직 이음부<Br> 최하단 내외부 300mm**|

> ※ 스테인리스강의 경우 용접 종료 후 PT 적용
