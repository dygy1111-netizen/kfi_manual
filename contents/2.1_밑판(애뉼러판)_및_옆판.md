### 시험개소 및 범위
| 구분 | 검사 종류 | 검사량(범위) |
|------|-----------|----------------|
| **애뉼러판 맞대기이음** | **① PT + MT**<br>초층 PT → 용접종료 MT | 전체 길이 |
|  | **② RT 또는 PAUT + PT/MT** | - 끝단 250mm 이내: 전체 중 **50% RT 또는 PAUT** <br>- 250mm 초과부: **PT + MT** |
| **밑판(애뉼러판 없음)** | **PT + MT** | - 옆판 최하단 중심 기준<br>→ **안쪽 200mm + 바깥쪽 전체 용접부** |
| **밑판 이음부(T자부·3매 겹침부)** | **PT + MT** | - T자부 방사선 불가 영역<br>→ 길이 **200mm × 3방향** |
| **밑판 일반 용접부(전 길이의 3%)** | **① MT** 또는 **② 진공시험** | - 전체 용접길이의 **3%**<br>- 단, **10m 초과 시 최대 10m** |
| **옆판 맞대기 용접부** | **① RT** 또는 **② PAUT** | 전체 용접 길이 |
|  | **PT + MT** (두께 < 6mm) | 전체 |
| **옆판 개구부 / 물받이(sump)** | **MT** | 전체 길이(용접 종료 후) |
|  | **PT(초층) + MT** | - 고장력강판 전두께<br>- 연강판 두께 >25mm |
| **개구부 맞대기용접부** | **RT 또는 PAUT** | 전체 길이 |
| **가설재 부착·제거 부위** | **VT 또는 MT** | 제거 부위 전체 |
|  | **MT 필수** | - 고장력강판<br>- 연강판 두께 >25mm |
| **기타 형상 탱크 용접부** | **RT / PAUT / MT / PT 중 선택** | 모든 용접부 |
| **RT·PAUT 곤란 시 대체** | **UT(초음파탐상시험)** | 해당 부위 |
