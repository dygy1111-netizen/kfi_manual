### 시험개소 및 범위

| 구분 | 검사 종류 | 적용 조건 / 검사 범위 |
|------|-----------|------------------------|
| **바. 옆판 최하단 개구부 및 물받이(sump) 용접부** | **MT(자기탐상시험)** | - 용접종료 후 전체 실시 |
| | **PT(침투탐상) — 초층용접 후** | - 옆판이 **고장력강판(≥500MPa)** → 모든 두께 적용<br>- **연강판 두께 25mm 초과** 시 적용 |
| **사. 개구부 및 물받이의 맞대기 용접부** | **RT 또는 PAUT** | - 용접종료 후 전체 길이 실시 |
| **아. 가설재 부착·제거 부위** | **VT 또는 MT** | - 가설재 제거 부위 전체에 대해 육안 또는 MT 검사 |
| | **MT 필수** | - 옆판이 **고장력강판**인 경우<br>- 연강판 **두께 25mm 초과**인 경우 |
| | **PT 대체 허용** | - MT 실시가 불가능할 때 PT로 대체 가능 |
